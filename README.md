## Installation

```bash
$ npm install

# mysql installation
$ npm install --save @nestjs/typeorm typeorm mysql2

# mongodb installation
$ npm i @nestjs/mongoose mongoose

# swagger installation
$ npm install --save @nestjs/swagger

# class-validator installation
$ npm i --save class-validator class-transformer
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
